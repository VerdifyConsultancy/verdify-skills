# Automation

`bin/verdify` implements the deterministic foundation of the lifecycle:

- target-project initialization and routing;
- schema validation;
- sprint skeleton creation;
- worktree creation, locking, role leases, runtime namespaces, inspection, review worktrees, and release;
- worker/critic prompt compilation with input hashes;
- GitHub label bootstrap, issue/PR snapshots, and contract reconciliation.

Use `bin/verdify --help` for commands. The CLI automates transitions and evidence collection, not product judgment, architecture approval, critic independence, or production authorization.
