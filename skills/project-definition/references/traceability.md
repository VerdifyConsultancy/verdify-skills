# Traceability

Minimum chain:

```text
source or approved decision
  -> requirement
  -> product workflow or design surface
  -> lifecycle coverage, relationship, or open gap
  -> architecture component/module
  -> sprint acceptance criterion
  -> lane criterion
  -> evidence
```

A missing link is either a gap or deliberate non-traceable context. Record exceptions explicitly. Lifecycle coverage marked `not_applicable` or `deferred` still needs a source, decision, or rationale. Never renumber stable IDs merely for presentation.
