# ADR-0002: Eight coherent lifecycle skills

- Status: accepted
- Date: 2026-06-22

## Decision

Preserve 17 detailed lifecycle stages but expose eight discoverable skills. Group discovery through design into `project-definition`, architecture and modules into `architecture-contracts`, plan and lane compilation into `sprint-planning`, implementation and closeout into `lane-delivery`, and integration through outcome into `release-verification`.

## Consequences

Activation is more reliable and context overhead is lower. Fresh-context criticism and deployment verification remain separate privileged roles.
